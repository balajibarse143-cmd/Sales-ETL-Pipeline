# Sales-ETL-Pipeline


##  Overview
This project demonstrates a simple ETL pipeline using PySpark.

##  Tech Stack
- PySpark
- Azure Data Factory (concept)
- CSV Data

##  Steps
1. Extract data from CSV
2. Clean data (remove nulls)
3. Transform (calculate total sales)
4. Aggregate (category-wise sales)
5. Load output

##  Run
```bash
python scripts/etl_pipeline.py
